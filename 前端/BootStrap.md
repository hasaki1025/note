# BootStrap

- ## BootStrap简介

  - BootStrap是依赖于JQuery的框架，内部带有写好的CSS样式

- ## 布局容器

  - BootStrap提供的布局主要两种，一种是两边留白，一种是占据全屏

  - (1)固定宽度并支持响应式

    ```html
    <div class="container">
        hello
    </div>
    ```

  - (2)采用100%宽度占据所有视口

    ```html
    <div class="container-fluid">
        hello
    </div>
    ```
